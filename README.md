# spring-boot-store-api
# 🛒 Store Management REST API

A REST API built with Spring Boot for managing a store backend.

## 🛠️ Tech Stack
- Java 21
- Spring Boot 3.4
- MySQL
- Spring Data JPA
- Flyway
- MapStruct
- Lombok

## 📌 API Endpoints
| Method | Endpoint     | Description    |
|--------|--------------|----------------|
| GET    | /users       | Get all users  |
| GET    | /users/{id}  | Get user by ID |

## 📚 What I Learned
- JPA Relationships (OneToMany, OneToOne, ManyToMany)
- DTO Pattern with MapStruct
- Flyway database migrations
- Clean REST API structure
